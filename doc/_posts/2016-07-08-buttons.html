Buttons

Combine button classes with style, size and semantic color classes to create a variety of buttons.

Color variations

Grey

Grey buttons are usually used for neutral actions such as copy, next, back, etc.

<button class="btn btn-default">Button</button>
<button class="btn btn-ghost">Button</button>
CTA color

CTA (Call To Action) buttons usually use brand color.

<button class="btn btn-default cta">Sign Up!</button>
<button class="btn btn-ghost cta">Sign Up!</button>
Red

Red buttons are usually used for critical or negative actions such as delete, stop, etc.

<button class="btn btn-default danger">Delete</button>
<button class="btn btn-ghost danger">Delete</button>
Green

Green buttons are usually used for positive actions such as submit, complete, ok, etc.

<button class="btn btn-default success">Submit</button>
<button class="btn btn-default success">Submit</button>
Yellow

Yellow buttons are not as critical as red buttons. They can be used to bring some attention.

<button class="btn btn-default warning">Continue</button>
<button class="btn btn-default warning">Continue</button>
Blue

Blue buttons are usually used for informational actions such as help, what's this, read more, etc.

<button class="btn btn-default info">Help</button>
<button class="btn btn-default info">Help</button>
